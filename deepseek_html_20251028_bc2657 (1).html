Project Introduction

This project is a smart energy-saving lighting system built using a microcontroller, integrating motion detection, ambient light sensing, and current monitoring to automate and optimize bulb usage.

It connects to Wi-Fi and communicates with Ubidots to publish real-time data such as light status, current consumption, and energy saved, while also allowing remote manual control. The system uses a motion sensor to detect motion, an LDR to measure ambient light, and a current sensor to monitor current flow.

Smart Light Control Circuit

This smart light control circuit is designed to automate lighting based on motion and ambient light conditions. It uses an ESP32 microcontroller to process sensor inputs and control relays that switch 12V lamps on or off.

The system integrates a PIR sensor to detect motion and a Light Dependent Resistor (LDR) to measure brightness. When motion is detected and ambient light is low, the ESP32 activates the corresponding relay to power the lamp.

3. Main Control Loop

void loop() {
  int motion = digitalRead(PIRPIN);
  int lightLevel = analogRead(LDRPIN);
  float current = readCurrent();
  
  if (forceOn) {
    for (int i = 0; i < 4; i++) {
      if (!bulbOn[i]) {
        controlBulb(i, true);
      }
    }
  } else {
    if (motion == HIGH && lightLevel < LIGHT_THRESHOLD) {
      for (int i = 0; i < 4; i++) {
        if (!bulbOn[i]) {
          controlBulb(i, true);
        }
      }
    } else {
      for (int i = 0; i < 4; i++) {
        if (bulbOn[i]) {
          controlBulb(i, false);
        }
      }
    }
  }
  
  delay(500);
}

4. Custom Functions

void controlBulb(int index, bool state) {
  int pins[] = {RELAY1_PIN, RELAY2_PIN, RELAY3_PIN, RELAY4_PIN};
  digitalWrite(pins[index], state ? HIGH : LOW);
  bulbOn[index] = state;
}

float readCurrent() {
  int raw = analogRead(CURRENTSENSORPIN);
  float voltage = (raw / 4095.0) * 3.3;
  float offset = 3.3 / 2.0;
  float sensitivity = 0.185;
  float current = (voltage - offset) / sensitivity;
  return abs(current);
}

Remote Monitoring – Feature Breakdown

Element of Ubidots Dashboard

Light Status (ON/OFF): Displays the current state of each connected lamp. Helps users know which lights are active at any moment.

Current Consumption: Shows real-time electrical current usage for each lamp. Useful for tracking power draw and identifying high-consumption devices.

Motion Detection: Uses a PIR sensor to detect movement in the monitored area. When motion is detected, the system activates the corresponding light automatically, enhancing security and convenience.

Light Detection: Utilizes an LDR (Light Dependent Resistor) to measure ambient light levels. When the surrounding light falls below a certain threshold, the system automatically turns on the lights to maintain visibility and comfort.

Manual Control Switches: Allows users to override automation and manually toggle lights on or off from the dashboard interface.

Cloud Features

  • Remote Manual Override: Enables users to control lighting from anywhere using a cloud-connected interface—ideal for travel or remote access.
  • Historical Data Analytics: Provides insights into usage patterns over time, helping optimize energy consumption and predict future needs.
  • Real-time Alerts: Sends notifications for unusual activity, such as unexpected power spikes or motion detection when no one is home.
  • Energy Efficiency Reports: Summarizes performance and savings, offering actionable recommendations to improve sustainability.
